# nvim-config
idk, what i have set up for vim/nvim rn

## Prerequisites
 - NVim
 - NodeJS
 - NPM
 - vim-plug
 - Jedi (for python support)
 - FZF (for fuzzy-finding)
 - fortune (for the start screen)

## Info
For the ton of extensions, use https://github.com/junegunn/vim-plug
